openMSX Manuals

We recently started converting the text format HOWTO into several guides in HTML format. The document titles that are hyperlinked are already available, the others will follow in the near future. For the topics not yet available in the new format, read openMSX-HOWTO.txt inside the openMSX tree (so, this link will only work if you browse this page from your local openMSX tree) or online.

Compilation Guide
This guide describes how you can get the openMSX sources and compile them.
Setup Guide
This guide describes how you can configure openMSX to emulate actual MSX machines. It also describes how you can have openMSX start up with your personal settings, how you can configure openMSX and your system for optimal performance and several other configuration related topics.
User's Manual
This manual describes all the things you can do with openMSX once it is fully running.
